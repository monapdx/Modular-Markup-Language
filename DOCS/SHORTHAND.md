## SHORTHAND RULES

- [ ] **Tags** **must** be **longer than four letters** to have a shorthand version assigned to them
- [ ] The **preferred length** for **all** shorthand versions of tags is **exactly three letters** whenever possible, **four** when *absolutely necessary*, and **never more**
- [ ] **All** shorthand versions of tags use **the first three** *or* **four letters** of the tag
- [ ] Some **longer tags** do **not** have a shorthand version due to **awkwardness of the first three letters** and **loss of meaning** exceeding what I find acceptable (example: **Timeline**)
- [ ] In *some cases*, **file extensions can be used as shorthand** for some Tags, but they *must be* **exactly three characters**
