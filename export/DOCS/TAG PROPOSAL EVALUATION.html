title TAG PROPOSAL EVALUATION
category checklist
What is the TAG being proposed? Is the value and/or uniqueness obvious immediately? Is it distinct enough from existing tags? Does it conflict with any existing tags? Will people likely struggle to understand it quickly and easily? Will it require child tags? Which existing tags would it nest within? Does it have self-contained meaning without needing other tags? Is it mainly a container for other tags? How many letters is the tag? Will creating a shorthand version be fairly easy or more difficult? How well do the first 3-4 letters of the word preserve enough of the original meaning? Have the first 3 letters of the tag already been assigned as a shorthand for an existing tag? If yes, have the first 4 letters?
